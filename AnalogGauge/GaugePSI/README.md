# 圓形儀表偵測 

1. 利用霍夫圓畫出角度得知最大最小值角度。此圖應是 45 ~ 315 度。
2. 將圓圖二值化 ➔ 霍夫線找出指針 ➔ 藉由角度找出數值。

問題 : 應用於其他圖上效果不佳。
